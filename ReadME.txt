Otwórz ten projekt w środowisku programistycznym Visual Studio Code. 
Jak to zrobić, otwórz Visual Studio Code, następnie w lewym górnym rogu kliknij przycisk 'Plik' , 
następnie wybierz 'Otwórz folder' z rozwijanego menu, 
znajdź folder z projektem i wybierz ten folder, po otwarciu projektu wywołujemy terminal. 
W terminalu wpisz polecenie „npm install”. 
Jeśli pojawi się błąd w terminalu, musisz zainstalować Node.js (wersja 18.17.1 LTS)
link: nodejs.org/en 
Uruchom ponownie komputer po instalacji, po zainstalowaniu node.js ponownie użyj polecenia „npm install” w terminalu. 
Aby uruchomić projekt, użyj polecenia „npm start” lub „npm run start” 
Szczęśliwego kodowania 😊
